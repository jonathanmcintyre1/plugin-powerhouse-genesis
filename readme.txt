Elegant content & image protection for WordPress websites. Prevent copying, right-clicking, and unauthorized use of your valuable content.

== Description ==

CopyProtect is a comprehensive solution to protect your valuable content and images from being copied, stolen, or misused online. With an intuitive admin interface and a wide range of protection options, CopyProtect makes it easy to safeguard your intellectual property.

= Key Features =

* **Text Protection**: Prevent text selection, copying, or dragging from your content
* **Image Protection**: Disable right-clicking and dragging of images, and implement advanced image protection techniques
* **Keyboard Shortcut Blocking**: Prevent common keyboard shortcuts used to copy content
* **Browser Interaction Controls**: Disable printing, view source, and implement HTML obfuscation
* **Customizable Messaging**: Configure user notifications and visual protection indicators
* **Granular Control**: Apply different protection settings per content type
* **Lightweight Performance**: Minimal impact on page load times and user experience

= Protection Methods Include =

* Disable right-clicking site-wide or on images only
* Prevent text selection and drag-and-drop operations
* Block keyboard shortcuts (Ctrl+A, Ctrl+C, Ctrl+X, Ctrl+S, Ctrl+U, F12)
* Add transparent overlays on images
* Serve images via CSS backgrounds to hide source URLs
* Prevent image hotlinking
* Anti-inspect tool options
* And many more features

== Installation ==

1. Upload the `copyprotect` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the CopyProtect menu in your WordPress admin
4. Configure protection settings as desired
5. Save your changes

== Frequently Asked Questions ==

= Will this plugin slow down my website? =

No, CopyProtect is designed to be lightweight and optimized for performance. It adds minimal overhead to your website.

= Is this plugin compatible with page builders? =

Yes, CopyProtect works with all major page builders including Elementor, Beaver Builder, Divi, and more.

= Can I exclude certain pages from protection? =

Yes, you can exclude specific pages, posts, or categories from protection using the Advanced Rules tab.

= Is this 100% effective against all copying methods? =

While CopyProtect implements many techniques to prevent copying, determined users with technical knowledge may find ways around these protections. However, it will effectively deter most casual attempts to copy your content.
